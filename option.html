My CSS Option Assignment

For this assignment, I chose some of the advanced CSS features and decided to create a travel website called Wander World. My website is all about exploring beautiful places around the world and encouraging people to discover and explore new destinations. I built it using HTML5 and CSS3, focusing on applying different CSS techniques that I learned from Chapters 6 and 7 of the Frain textbook.

My website includes three pages: a Home page, a Gallery page, and an About page. The Home page welcomes visitors with a large background image and a transparent overlay that introduces what the site is about. The Destination page has a horizontal scrolling section where visitors can look through different travel photos with cool visual effects. The About page explains the purpose of the site and shares a few travel tips that are displayed in two columns to make it easier to read.

Throughout the site, I used several CSS features to make it look more modern and visually appealing. These include shadows, filters, transparency effects, and a media query for high-resolution images. My main goal was to design a simple but creative website that shows I've learned about CSS styling and layout while keeping the theme fun and travel-focused.

CSS Features Used

  1. Multi-Column Layout: I used the multi-column layout feature on my About page (about.html) to display the Our Story and Our Mission sections side by side. This makes the text easier to read and creates a balanced visual layout. The CSS property used was column-count: 2; to divide the content into two columns.
  2. Horizontal Scrolling: I applied horizontal scrolling on my Destinations page for the travel destination scrolling cards. This allows users to scroll sideways to view more destination cards smoothly. The CSS properties display: flex; and overflow-x: auto; were used to create this scrolling effect. I also added some responsive media styles to make sure the gallery is more well organized for mobile users which has been approved when it was tested.
  3. Alpha Transparency: I used alpha transparency in my Home page (index.html) hero section overlay behind the main title and tagline. This effect was achieved using the RGBA color value rgba(0, 0, 0, 0.6), which creates a semi-transparent background while keeping the text readable over the image.
  4. Linear Gradients: On my Home page, I used a subtle linear gradient on the hero banner overlay to blend the background image and make the text stand out. The gradient was created using the CSS property background: linear-gradient() with soft blue tones that match the site's theme.
  5. @Font-Face Rules: I imported a custom font for my headings using the @font-face rule in my CSS. I was able to use two fonts from Google Fonts like Montserrat and Pacifico, and applied them both across all pages to make the titles and hero text match the travel brand's style and stand out more. The custom fonts also helps maintain design consistency throughout the site.
  6. Text and Box Shadows: I used text shadows and box shadows throughout the site to add depth and emphasis to key elements. For example, text shadows were applied to the hero heading on the Home page to improve visibility against the image background, and box shadows were used around gallery boxes on the Destination page to give a 3D appearance and higglight as a transforming appearance.
